# Конспекты по дисциплинам "Математический анализ" и "Специальные главы высшей математики"

## Разделение на дисциплины
- **`special_chapters`** - спец главы ВМ
- **`math_analysis`** - математический анализ (на данный момент в репозиории отсутствует из-за отсутствия лекций)

## PDF и Typst файлы
- `.pdf` файлы для собственно удобного просмотра
- `.typ` файлы как source файлы для pdf-ок, любые изменения и предложения для изменений должны быть обращены на `.typ` файлы

## Изображения-референсы
В `src_images` лежат исходные изображения, с которых написаны файлы `.typ`. Они не отсортированы и служат источником, откуда берется весь конспект, стараемся ничего не придумывать

## Добавления и правки
Чтобы добавить исходники (ну или сразу файлы Typst, если вы такие молодцы), делайте пулл (следите за тем, что добавляете туда куда нужно)\
Если увидели ошибку, то используйте таб `Issues` с скринами-подтверждениями доски
